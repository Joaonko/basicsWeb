Meu Primeiro Site com Bootstrap

Bootstrap é um framework front-end gratuito e de código aberto que utiliza HTML, CSS e JavaScript para facilitar a criação de sites e aplicações web responsivos, ou seja, que se adaptam a diferentes tamanhos de tela. Ele fornece uma coleção de componentes de interface prontos, como botões, formulários, grades e modais, que aceleram o desenvolvimento e garantem consistência visual e um design consistente.
